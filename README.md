## 💾 Commit Convention

---

### 💬 Commit Message

| 타입 | 설명 |
|------|------|
| **CHORE** | 코드 수정, 내부 파일 수정 |
| **FEAT** | 새로운 기능 추가 |
| **FIX** | 버그 및 오류 수정 |
| **DOCS** | README 등 문서 수정 |
| **REFACTOR** | 전면 수정(코드 리펙토링) |
| **TEST** | 테스트 코드 추가 및 수정 |

### 예시

```bash
FEAT: 로그인 기능 추가
FIX: 회원가입 시 이메일 검증 오류 수정
DOCS: README 사용법 추가
REFACTOR: 사용자 서비스 로직 개선
TEST: 로그인 기능 테스트 코드 작성
